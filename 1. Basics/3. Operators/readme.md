_Operators_
==

### _Arithmetic Operators_    

  
BINARY OPERATORS :

- Exponent - (**)

- Multiplication - (*)

- Division (/)

- Floor Division (//)

- Modulo (%)

- Addition (+)

- Subtraction (-)

UNARY OPERATORS :

- Unary plus (+)

- Unary minus (-)

PRIORITY :

Expression :  
```
10 - 6 ** 2 / 9 * 10 + 1
```

1st priority - '**'  

```
10 - 36 / 9 * 10 + 1
```


2nd priority - '/'

```
10 - 4 * 10 + 1
```
3rd priority - '*'

```
10 - 40 + 1
```

4th priority - '-'

```
-30 + 1
```

5th priority - '+'

```
-29
```
