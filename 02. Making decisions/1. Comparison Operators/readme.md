_Comparison Operators_
==

- Equal to (==)

    ```python 
    print(2==3)
    ```

    >False

- Not equal to (!=)

    ```python
    print(2!=3)
    ```

    >True

- Greater than (>)

    ```python
    print(2>3)
    ```

    >False

- Greater than or equal to (>=)

    ```python
    print(2>=3)
    ```

    >False

- Lesser than (<)

    ```python
    print(2<3)
    ```

    >True

- Lesser than or equal to (<=)

    ```python
    print(2<=3)
    ```

    >True
